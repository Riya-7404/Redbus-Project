# 🚌 RedBus Clone

## 📖 Project Overview

RedBus Clone is a full-stack web application developed using Java, React, and MySQL. It allows users to search buses, book tickets, manage bookings, and securely log in.

---

## 🚀 Features

- User Registration & Login
- Search Available Buses
- Ticket Booking
- Booking History
- Cancel Booking
- Admin Dashboard
- Responsive User Interface

## ⚙ Installation

1. Clone the repository
2. Install frontend dependencies

```
npm install
npm start
```

3. Run Spring Boot backend

4. Import MySQL database
